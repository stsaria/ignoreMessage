No reply chat is no need.

There are no rules. No categorized content (in other words, just chat). You can write anything. But if there is no reply, it will be considered meaningless and the post will be deleted.

Why need respond?

This is because many people don't really want to reply to posts that have gone unanswered for a long time (an easy example is comments on YouTube). So we consider unanswered posts to be unavoidable, but wasteful.

No response means you have been abandoned. This is the chat algorithm that I find most efficient.
